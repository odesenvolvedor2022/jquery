Aula 02 - Sintaxe do jQuery

AULA ANTERIOR PRÓXIMA AULA

Nesta aula do curso de jQuery vamos aprender como se escreve os códigos, ou seja, a sintaxe do jQuery. Você vai perceber que ele simplifica bastante a escrita e pode acelerar o processo de codificação.

Pegando um elemento no Javascript puro:

No exemplo abaixo, iremos pegar um elemento <p> com Javascript puro e pedir para preenche-lo com a palavra teste:

< p  id = "teste" > </ p >


<script>


document.getElementById("teste").innerHTML = "teste";


</script>

Pegando um elemento no Javascript puro:

No exemplo abaixo, iremos pegar um elemento <p> com jQuery e pedir para preenche-lo com a palavra teste:

< p  id = "teste" > </ p >


<script>


$ ( "teste" ) . html ( "teste" );


</script>

Add class a um elemento com jQuery:

No exemplo abaixo, iremos adicionar uma classe a um elemento <p>:

< p  id = "teste" > </ p >


<script>


$ ( 'p' ) . addClass ( 'nome-da-classe' );


</script>
