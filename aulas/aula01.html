Aula 01 - Aprendendo a instalar o jQuery no projeto

INÍCIO PRÓXIMA AULA
  1. Acessar o site do jQuery;
  2. Fazer o download do arquivo;
  3. Salvar na pasta do projeto;
  4. Adicionar o script do jQuery antes do fechamento da tag body:

< body >

< script  src = "js/jquery-3.6.1.js" > </ script >

< /body >
